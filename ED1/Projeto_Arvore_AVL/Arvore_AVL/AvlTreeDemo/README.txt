 Objetivos: O trabalho realizado consiste de uma implementação de uma árvore AVL para armazenamento de palavras obtidas a partir de arquivos textos, simulando um índice.



Para Compilação:

 - Em Linux:	Basta rodar o script make.sh
 - Em Windows:	Basta definir o local do g++ no arquivo make.bat e executá-lo.


Para Testes:

  - Em Windows:	Executar bin/run tests.bat
  - Em Linux:	Executar bin/run tests.sh



Todo código deste arquivo é dedicado ao domínio público.
